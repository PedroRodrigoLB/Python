# Organizando uma lista de compras
- Objetivo é criar um programa em python que permite ao usuario adicionar itens à lista, remover itens, exibir a lista completa e sair do programa.

# Requisitos:

    1 - Menu Principal
        - Adicionar item à lista de compras;
        - Remover item da lista de compras;
        - Exibir todos os itens da lista de compras;
        - Sair do programa.
    
    2 - Adicionar Item:
        - O usuário pode digitar o nome de um item para adicionar à lista;
        - Se o item já estiver na lista, o programa deve avisar e não adicionar o item duplicado.

    3 - Remover item:
        - O usuário pode remover um item da lista digitando o nome do item;
        - Se o item não estiver na lista, o programa deve avisar que o item não foi encontrado.

    4 - Exibir a lista completa:
        - O programa deve exibir todos os itens atualmente na lista, um por linha;
        - Se a lista estiver vazia, o programa deve informar que não há itens na lista.

    5 - Sair:
        - O programa deve encerrar a execução quando o usuário escolher a opção de sair.
